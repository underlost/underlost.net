Jaded Gamer

Jaded Gamer is a view game news site aggregator, run and managed completely in the cloud, utilizing various SAAS applications. RSS updates are pushed to the site via the pubsubhubbub protocol. New stories are then indexed immediately, and can be searched through by the elastic-search based server. For additional details, you can learn more on Medium.
